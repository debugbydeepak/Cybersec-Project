Getting Started with SECUREWAY

Welcome to the official SECUREWAY documentation. Our autonomous agent is designed to integrate seamlessly into your existing CI/CD pipelines and development workflows, identifying complex business logic vulnerabilities before they reach production.

To get started, you'll need an active API key from your dashboard and the SECUREWAY CLI client.

Installation

The SECUREWAY CLI is distributed as a standalone binary for macOS, Linux, and Windows, or as a Docker container.

Using cURL (macOS / Linux)

curl -sSL https://secureway.dev/install.sh | bash

Using Docker

docker pull secureway/scanner-agent:latest

Configuration

Authenticate your agent by providing your API key. You can do this via an environment variable.

export SECUREWAY_API_KEY="sw_live_xxxxxxxxxxxxx"

You can also configure the agent using a secureway.yml file placed in your project root.

target_url: "https://staging.example.com" max_depth: 3 modules: - boolean_logic - idor - ssrf headers: - "Authorization: Bearer test_token"

Running Scans

Initiate a scan directly from your terminal or CI environment.

secureway scan --target https://staging.example.com --report-format json

The agent will autonomously map the application, execute business logic checks, and output a detailed report summarizing the findings.
